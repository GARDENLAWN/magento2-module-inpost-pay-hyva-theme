# InPost Pay for Magneto 2
Module adds integration of customer and guest carts with InPost Pay.
Compatibility with Hyva theme.

### Compatibility
- Magento Community 2.4.6
- Not tested in other versions

### Installation
To enable this extension run following commands:
- `bin/magento module:enable InPost_InPostPayHyvaTheme`
- `bin/magento setup:upgrade`
